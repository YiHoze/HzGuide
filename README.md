# HzGuide

HzGuide is a LaTeX class for manuals such as user guides and installation instructions.
This class is supposed to be used with XeLaTeX or LuaLaTeX.
The key features is as follows:

* Supports layouts for mobile phones and slides as well as convetional ones.
* Provides various macros for images. Using them, users can globally or locally set up several options for images, such as size, position, and frame.
* Provides several bounded boxes. Users can create a new environment that uses one of them.
* Users can create a new definition list with ease.
* Provides many of miscellaneous tools.
